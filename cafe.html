Cafe Bookings & Info

There are limited seating within the Cafe so for that reason if you would like to sit in it is recomended that you book a seat for your party in advance. There is plenty of space and seating outside if you dont book a table. We would apreciat it if you have booked a table and cant make it, if you would kindly cancle with us so we can open that booking for another party. It not necessary to book a table as we will offer you a table if one is free on your arrival.
